## JUCE

My brand-new plugin I'm creating is titled *Vox Genesis*. *Vox* being latin for voice, and *genesis* representing the start of something new. In this case, It's my journey in applying my C++ knowledge to something real. I have really enjoyed using synthesizers in my DAW of choice, Ableton, and so finding out there is a framework written in C++ feels like it is tailor-made for me. The Projucer is a program that generates the necessary files for you to just get started writing code in your IDE. I chose Visual Studio 2026. 

For now, this is a single oscillator with an ADSR envelope. The oscillator can generate square, sine, and triangle waves. The envelope is controllable with sliders in the UI. For anyone familiar with synths, this is pretty underwhelming. My plan is to add more oscillators and filters. I have no idea where this will take me, but I'd love to create a synth based on a new concept of some kind. I've considered creating a synth that uses some sort of generative approach, like taming an ecosystem from the outside.

So far, I have been using the excellent tutorials on The Audio Programmer's YouTube channel to get my feet wet and understand the framework more deeply. I'm about ready to dive in on my own and do some more DSP stuff. Stay tuned to see what I come up with. Stay curious, my friends.
